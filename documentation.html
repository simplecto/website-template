Introduction


It is easy to get started with rubber ducking. If you use any of the IntelliJ products of Visual Studio Code, then we have you covered.

If you are not familiar with Rubber Duck Debugging, then we will let Wikipedia do the work there. They have an excellent overview. Go read that then come back when you are ready.


How does it work?


Get stuck? Summon the duck!

Whenever you get stuck or have doubts when coding, you can “summon the duck” for a quick chat. The idea here is that you will talk to him (he does not say much), and in doing so you will arrive to some kind of clarity.

That, “Ah ha” moment.

Supported apps We aim to support the following Intellij IDEs:

  • AppCode

  • Intellij IDEA
  • PhpStorm
  • PyCharm
  • Rider
  • RubyMine
  • WebStorm

You can install the plugin directly from the plugin manager in IntelliJ. Simply open the preferences panel > Plugins > Marketplace > (search rubber duck)

You will see Rubber Duck in the options. Click install and it will download. NOTE: You might have to restart your IDE for it to take efect.

How do I summon the duck?

Easy! There are pre-defined short-cut keys.

On Apple Shift-Command-D, or use Tools > Summon Duck

On PC and Linux Shift-Ctrl-D, or use Tools > Summon Duck

You can also dismiss Duck with the same command.
